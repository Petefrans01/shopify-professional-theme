# Thème Shopify Professionnel

Ce projet contient un thème Shopify moderne et professionnel avec toutes les fonctionnalités essentielles.

## Structure du Projet

```
theme/
├── assets/          # CSS, JS, images
├── config/          # Configuration du thème
├── layout/          # Layouts principaux
├── locales/         # Traductions
├── sections/        # Sections réutilisables
├── snippets/        # Composants réutilisables
└── templates/       # Templates de pages
```

## Installation

1. Installez Shopify CLI:
```bash
npm install -g @shopify/cli @shopify/theme
```

2. Connectez-vous à votre store:
```bash
shopify auth login
```

3. Développement local:
```bash
shopify theme dev
```

4. Déploiement:
```bash
shopify theme push
```
